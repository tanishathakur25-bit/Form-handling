# User Details Form

## 📌 Description

This is a simple User Details Form built using HTML, CSS, and JavaScript.
It takes input from the user and displays the entered data on the screen after form submission.

## 🚀 Features

* User-friendly form interface
* Takes multiple inputs (Name, Age, Place, etc.)
* Displays entered data dynamically
* Clean and simple design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM)

## 📂 Project Structure

* `index.html` → Structure of the form
* `style.css` → Styling of the page
* `script.js` → Form handling and output display

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Enter details and click Submit

## 📸 Output

Displays user details like:

* Name
* Age
* Place
* Father Name
* Hobby
* And more

## 💡 Learning

* Learned form handling using DOM
* Learned how to take user input
* Learned how to display data dynamically
